
## Installation

Install my-project with git

#### Clone Project
```bash
  git clone https://github.com/ruddesh/loop-backend-api.git loop-backend-api
  cd loop-backend-api
```
#### change directory
```bash
  cd loop-backend-api
```
#### sail
```bash
  composer require laravel/sail --dev
```
#### sail composer
```bash
  php artisan sail:install
```
#### generate key
```bash
  php artisan key:generate
```
